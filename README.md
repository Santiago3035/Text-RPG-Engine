## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/Text_RPG_Engine.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Text_RPG_Engine
   ```
3. Install dependencies (if any):
   ```sh
   pip install -r requirements.txt
   ```
4. Run the application:
   ```sh
   python main.py
   ```
